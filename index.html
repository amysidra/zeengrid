Zeen grid

absolutely clean & simple grid

Spectacularly Easy Responsive Design The Responsive Grid System isn't a framework. It's not a boilerplate either. It's a simple, easy & flexible way to create a responsive web site.

Responsive

ZEENGRID elegantly scales down to downsized browser windows, tablets, mobile phones. So people visiting your site will receive a layout that's tuned to the size of their browser window. Go ahead, resize this page! Say goodbye to horizontal scrollbars.

Smart

Most grid systems use one class for all units of a single size, such as .alpha .omega .first .end and .last. With ZEENGRID each zeen knows whether it's first, middle, or last in its respective row; no extra classes needed, so smaart! isn't?

Easy

ZEENGRID made by using ruby, works with either SASS or COMPASS and it could plugs into your existing HTML and CSS, and it's so easy to do so. ZEENGRID is your friend. Use it on as simple or as complex a project as you wish. You'll be done in minutes.

How to use it?

SStart with the .wrap for wrapper. Inside the wrapper Create divs with a class of .row. Inside each row, create zeen for content. Apply the associated .zeen-# (1-12) class , and İşte!
